# DRL-Pytorch 学习仓库 — 使用须知

## 1. 环境激活 (conda)

本项目使用 conda 环境 **rl_learning**，基于 Python 3.11.5 + PyTorch 2.1.0。

```bash
# 激活环境（每次登录后第一条）
conda activate rl_learning

# 验证环境
python -c "import torch; print(f'PyTorch {torch.__version__}, CUDA {torch.cuda.is_available()}')"

# 退出环境
conda deactivate
```

常用 conda 命令：

```bash
conda env list          # 查看所有可用环境
conda list              # 查看当前环境已安装的包
pip list | grep torch   # 查看 torch 相关包版本
```

## 2. Git 分支管理

仓库有 3 个分支，对应不同成员：

| 分支  | 成员 | 说明       |
|-------|------|------------|
| main  | 公共 | 上游同步   |
| tmq   | 唐明清 | 个人作业 |
| hzm   | 黄志铭 | 个人作业 |

```bash
# 查看当前分支和状态
git status
git branch -a

# 切换到自己的分支（示例）
git switch tmq       # 唐明清
git switch hzm       # 黄志铭

# 拉取最新远端代码
git pull origin <branch-name>
```

## 3. 作业流程

### 3.1 首次使用（在自己的分支上）

```bash
cd ~/Workplace/robot/DRL-Pytorch_learning
conda activate rl_learning
git switch <你的分支>        # 例如 git switch tmq
```

### 3.2 日常提交作业

```bash
# 1. 进入仓库
cd ~/Workplace/robot/DRL-Pytorch_learning

# 2. 激活环境
conda activate rl_learning

# 3. 确认在自己的分支
git branch

# 4. 写代码 / 改代码 ...

# 5. 提交
git add .
git commit -m "feat(<算法名>): <做了什么的简短描述>"
git push origin <你的分支>
```

### 3.3 运行某个算法

```bash
conda activate rl_learning
cd ~/Workplace/robot/DRL-Pytorch_learning/<算法目录>
python main.py
```

例如运行 PPO-Discrete：

```bash
cd ~/Workplace/robot/DRL-Pytorch_learning/3.1\ PPO-Discrete
python main.py
```

## 5. 注意事项

- **不要**在 `main` 分支上直接写代码。主分支用于同步上游更新。
- 提交前确认 `git branch` 显示的是你自己的分支。
- 如果依赖包缺失，用 `pip install <package>` 安装到 rl_learning 环境。
- GPU 服务器上有 NVIDIA 显卡，确保 PyTorch 能识别 CUDA。
